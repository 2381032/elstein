Testimony

Belajar web programming telah memberikan banyak hal berharga bagi saya. Awalnya, HTML tampak rumit, tetapi setelah mencoba dan mempraktikkannya, saya menyadari bahwa sebenarnya HTML tidaklah sesulit yang saya bayangkan. Hal yang sama berlaku untuk CSS meskipun menantang, ternyata cukup seru untuk dipelajari. Saya juga menemukan bahwa menggunakan Tailwind cukup menyenangkan, terutama saat mengubah gaya dari file eksternal menjadi inline. Sementara JavaScript mungkin bukan favorit saya, saya tetap termotivasi untuk memahaminya lebih dalam karena ilmu yang didapat dari mempelajarinya sangat berharga.
